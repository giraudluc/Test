Bar graph

Syntax 

''' plotly
{
  "data": [
    {
      "x": ["giraffes", "orangutans", "monkeys"],
      "y": [20, 14, 23],
      "type": "bar"
    }
  ]
}
'''

Bar

{ "data": [ { "x": ["giraffes", "orangutans", "monkeys"], "y": [20, 14, 23], "type": "bar" } ] }

Pie

Syntax 

''' plotly
{
  "data": [
    {
      "values": [19, 26, 55],
      "labels": ["Residential", "Non-Residential", "Utility"],
      "type": "pie"
    }
  ]
}
'''

{ "data": [ { "values": [19, 26, 55], "labels": ["Residential", "Non-Residential", "Utility"], "type": "pie" } ] }
